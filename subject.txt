Chapter I
Introduction
This topic is intended to introduce you to system administration. It will make you aware
of the importance of using scripts to automate your tasks. For that, you will discover
the "docker" technology and use it to install a complete web server. This server will run
multiples services: Wordpress, phpMyAdmin, and a SQL database.
__________________________________________________________________________________________
Глава I
Введение
Эта тема предназначена для знакомства с системным администрированием. 
Это позволит вам осознать важность использования скриптов для автоматизации ваших задач. 
Для этого вы откроете для себя технологию «докер» и будете использовать ее для полной установки веб-сервера. 
Этот сервер будет запускать несколько служб: Wordpress, phpMyAdmin и базу данных SQL.
__________________________________________________________________________________________
Chapter II
General instructions
• You must place all the necessary files for the configuration of your server in a folder
called srcs.
• Your Dockerfile file should be at the root of your repository. It will build your
container. You can’t use docker-compose.
• All the necessary files for your WordPress website should be in the folder srcs.
__________________________________________________________________________________________
Глава II
Основные инструкции
• Вы должны поместить все необходимые файлы для конфигурации вашего сервера в папку с именем srcs.
• Ваш файл Dockerfile должен находиться в корне вашего хранилища. Это создаст ваш контейнер. Вы не можете использовать docker-compose.
• Все необходимые файлы для вашего сайта WordPress должны находиться в папке srcs.
__________________________________________________________________________________________
Chapter III
Mandatory part
• You must set up a web server with Nginx, in only one docker container. The
container OS must be debian buster.
• Your web server must be able to run several services at the same time. The services
will be a WordPress website, phpMyAdmin and MySQL. You will need to make
sure your SQL database works with the WordPress and phpMyAdmin.
• Your server should be able to use the SSL protocol.
• You will have to make sure that, depending on the url, your server redirects to the
correct website.
• You will also need to make sure your server is running with an autoindex that must
be able to be disabled.
__________________________________________________________________________________________
Глава III
Обязательная часть
• Вы должны настроить веб-сервер с Nginx только в одном док-контейнере. Контейнерная ОС должна быть запущена Debian.
• Ваш веб-сервер должен иметь возможность запускать несколько служб одновременно. Сервисами будут веб-сайт WordPress, phpMyAdmin и MySQL. Вам нужно убедиться, что ваша база данных SQL работает с WordPress и phpMyAdmin.
• Ваш сервер должен иметь возможность использовать протокол SSL.
• Вы должны будете убедиться, что, в зависимости от URL, ваш сервер перенаправляет на правильный веб-сайт.
• Вам также необходимо убедиться, что ваш сервер работает с автоиндексом, который должен быть отключен.
