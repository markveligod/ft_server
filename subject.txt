Этот раздел предназначен для того, чтобы познакомить вас с системным администрированием. Это заставит вас осознать
о важности использования скриптов для автоматизации ваших задач. Для этого вам предстоит открыть
технология "docker" и использовать ее для установки полного веб-сервера. Этот сервер будет работать
несколько сервисов: Wordpress, phpMyAdmin и база данных SQL.
-------------------------------------------------------------------------------------
Вы должны поместить все необходимые файлы для настройки вашего сервера в папку под названием src.
• Ваш файл Dockerfile должен находиться в корне вашего репозитория. Он будет строить ваш контейнер. Вы не можете использовать docker-compose.
• * Все необходимые файлы для вашего сайта WordPress должны находиться в папке src.
--------------------------------------------------------------------------------------
Вы должны настроить веб-сервер с Nginx, только в одном контейнере docker. Контейнерная ОС должна быть debian buster.
• Ваш веб-сервер должен иметь возможность запускать несколько служб одновременно. Услуги сайта WordPress, phpMyAdmin и MySQL. Вам нужно будет убедиться, что ваша база данных SQL работает с WordPress и phpMyAdmin.
• Ваш сервер должен иметь возможность использовать протокол SSL.
• Вы должны будете убедиться, что, в зависимости от url-адреса, ваш сервер перенаправляет на правильный веб-сайт.
• * Вам также нужно будет убедиться, что ваш сервер работает с автоматическим индексом, который должен быть отключен.
